# Claude Marketplace for Dagster Plugins

A collection of Claude Code plugins for working with Dagster.

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add cmpadden/claude-plugins-dagster
```

Install a plugin:

```
/plugin install dg-troubleshoot@claude-plugins-dagster
```

## Available Plugins

**dg-troubleshoot** - Debug failing Dagster runs by fetching and analyzing run logs
